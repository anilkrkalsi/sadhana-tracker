# Sadhana Tracker landing page

Static, GitHub Pages-ready promotional website for **Sadhana Tracker: Meditation**.

## What is included

- Plain semantic HTML and responsive CSS
- No build process, JavaScript framework, cookies or analytics
- Official Google Play listing links and badge
- App icon and six optimized WebP screenshots retrieved from Google Play
- Open Graph image, favicons, manifest, robots file and sitemap
- Accessible native HTML FAQ controls

No `script.js` is included because the page uses native `<details>` FAQ controls and does not require JavaScript.

## Publish in exactly three steps

1. Create a **public** GitHub repository named `sadhana-tracker` under the GitHub account `anilkrkalsi`.
2. Extract the ZIP and upload **the contents of the extracted folder** to the repository root so that `index.html` is at the top level; commit the files to the `main` branch.
3. Open **Repository Settings → Pages**, select **Deploy from a branch**, choose `main` and `/ (root)`, then click **Save**. After deployment, open `https://anilkrkalsi.github.io/sadhana-tracker/`.

## Local check

You can open `index.html` directly, or serve the folder from any simple static web server. No installation or compilation is required.
